# 📊 Data Science Track

Learn to analyse, visualise, and communicate insights from data.

## Topics
1. **pandas & Polars** — DataFrames, method chaining, window functions, efficient dtypes
2. **NumPy** — arrays, broadcasting, vectorisation, linear algebra
3. **Visualisation** — matplotlib, seaborn, Plotly, interactive charts
4. **Statistics & EDA** — hypothesis testing, distributions, ydata-profiling
5. **SQL from Python** — DuckDB, SQLAlchemy, ibis

## Requirements
```bash
uv pip install -r requirements.txt
```

## Recommended order
01 → 02 → 03 → 04 → 05 → Project P1 (Polars benchmark)
