# P3 — RAG Chatbot over your Documents

## Goal
Build a chatbot that answers questions about your CV, GitHub READMEs, and notes using Retrieval-Augmented Generation.

## What you'll build
- Document loader (PDF + Markdown support)
- Chunking and embedding pipeline
- Vector store (ChromaDB)
- Retrieval + generation chain
- Streamlit UI

## Stack
- `anthropic` or `openai` for generation
- `chromadb` for vector storage
- `langchain` for orchestration
- `streamlit` for UI
- `pypdf` for PDF parsing
