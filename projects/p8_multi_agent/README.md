# Multi-Agent Data Analysis System

## Goal
LangGraph-based system with specialised agents: DataCleaner, StatisticsAnalyst, Visualiser, and ReportWriter. Given a CSV, the system autonomously produces a full analysis report.

**Key skills:** LangGraph, Claude tool use, Plotly, code execution, agent state management

## Structure
```
p8_multi_agent/
├── starter/    ← skeleton code with TODOs
└── solution/   ← complete reference implementation
```

## Setup
```bash
cd p8_multi_agent/starter
pip install -r requirements.txt
```

## How to work on it
1. Read this README fully
2. Start with `starter/` — fill in the TODOs in order
3. Run tests along the way: `pytest starter/`
4. When done (or stuck), compare with `solution/`
